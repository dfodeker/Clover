# Clover 
Terminal Based Notes Application
Clover is a simple terminal-based notes application that allows users to create, view, and manage their notes directly from the command line. It is designed to be lightweight and easy to use, making it perfect for quick note-taking tasks.

## Features
- Create, view, edit, and delete notes
- Organize notes by categories
- Search notes by keywords

## Installation
To install Clover, clone the repository and run the setup script:
```bash
git clone
```

cd clover
```bash
./setup.sh
```

## Usage
To start using Clover, run the following command in your terminal:
```bash
clover new [--title ""] [--tags a,b] [--stdin]
```

```bash 
clover list [--tag tagname] [--search "keyword"]
```

``` bash
clover view [note_id]
```
